    API Reference¶

    The X-Ray engine exposes a set of global objects and functions to Lua scripts. This page is an index of everything documented here, with frequency counts from analysis of 50 community mod repositories.

    How to read this reference

    Function signatures use the format object:method(param: type) -> return_type. Parameters in [brackets] are optional. Frequency counts show how many repos in the analysis used a given call — use this as a rough guide to what's important to learn first.
